# Architecture Summary

## Flask
Acts as:
- platform brain
- data source
- AI routing layer
- business/event/zone/map backend

## Unity
Acts as:
- immersive client
- player movement layer
- world hub
- portal navigation layer
- mobile-friendly playable surface

## Unreal later
Acts as:
- premium cinematic client
- high-end visual world layer

## AR / VR / MR later
Acts as:
- immersive display modes
- holographic overlays
- room-scale interaction
